## Build & Deploy Pipeline

**Scripts (from `package.json`):**
```bash
pnpm dev                    # Development server (with dns-fix.js)
pnpm build                  # Static export build (output: 'export' -> out/)
pnpm start                  # Serve production build
pnpm lint                   # ESLint
pnpm deploy                 # Full pipeline: prepare-live-env -> build -> FTP upload
pnpm deploy:ftp             # FTP upload only (out/ to remote)
pnpm deploy:prepare         # Generate .env.production.local from conf/start.md
pnpm monitor                # Check live frontend + backend health
pnpm monitor:json           # Health check in JSON format
pnpm monitor:insecure       # Health check skipping TLS verification
```

**Deploy Flow (`scripts/deploy.mjs`):**
1. `scripts/prepare-live-env.mjs` - Reads `conf/start.md` for backend URL, writes `.env.production.local`
2. `pnpm build` - Next.js static export to `out/`
3. `scripts/deploy-ftp.mjs` - Uploads `out/` via FTP/FTPS to remote host

**Production Monitoring (`scripts/monitor-production.mjs`):**
- Reads `conf/start.md` for frontend/backend URLs
- Checks frontend reachability and content
- Validates WordPress REST API (types endpoint, posts, front-page)
- Reports JSON or human-readable status
